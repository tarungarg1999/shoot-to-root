Let’s start with our Wifi Hacking Technique!

Wireless Network we can encounter:

1. WPA2 PSK (Personal)
2. WPA2 Enterprise

Here we are attacking WPA2 PSK (Pre Shared Key) which is basically called personal network or small level network or Medium level network than Enterprise.

Steps/Phases for attack WPA2 PSK:

Step 1-Placing Wireless card into monitor mode.
• Killing the current running process using tool airmon-ng .



• Starting wifi in monitor mode.



Step 2 – Discover information about the network.
• By using airodump-ng for wifi in monitor mode using command
==>airodump-ng wlan0mon
• Note down the BSSID and channel no. of ESSID-”Info Tech”.
• BSSID is nothing, its only as MAC address to identify the device.



Step 3- Select the network and capture data.
• Use this command to capture particular network.
==>airodump-ng -c 11 --bssid D8:32:E3:25:8C:6D -w capture wlan0mon
• Here -c is for channel and -w is for output file.



Step 4- Perform Deauth Attack.
• To generate Traffic and start more data flow over the network using the tool called aireplay-ng.



• Here -0 is to set deauth attack and 1 for setting up thread for the one time.



Step 5- Capture WPA handshake.
• This is basically four way handshake to get wireless password as hashes.



Step 6- Attempt to crack the handshake.
• We can use any tool to crack but here we are using aircrack-ng.
• We are using rockyou.txt provided by kali linux inbuilt for bruteforcing .
